Site Report

Overview

My whole web development learning journey is chronicled on this website. I explored fundamental concepts like HTML, CSS, layout design, version control, debugging, and responsive development while creating the portfolio. The project reflects the skills I have gained and how I applied them to build a clean and functional website.

Learning HTML and CSS

Any website's structural framework is provided by HTML (HyperText Markup Language), and its styling and presentation are handled by CSS (Cascading Style Sheets).

This module taught me about:
• HTML elements, attributes, and tags
• HTML5 semantic structure
• CSS properties, layout strategies, and selectors
• Modern HTML5 and CSS3 features

I used these concepts to design a neat, organized, and visually appealing layout for every page in the portfolio.

Learning Git and GitHub

Git is a distributed version control system used to track and manage code changes, while GitHub is an online repository hosting platform.

Commands I practiced:
• git clone
• git add
• git commit
• git push
• git pull

I also learned about branching, repository management, and writing proper commit messages. My project files were pushed to GitHub for version control.

Design Layout

My design focuses on simplicity and professionalism. To achieve this:
• I used a minimal color palette.
• Hover effects and transitions improve UX.
• Structured header and clean typography.
• Smooth internal navigation using anchor links.

Navigation Bar

The navigation bar appears on all pages for a consistent experience. Features:
• Logo
• Home, Projects, Contact, Site Report links
• Flexbox for alignment
• Clean and responsive layout

Home Page

The homepage introduces me and my work, including:
• Introduction text
• Profile image
• CTA button leading to Projects page

Flexbox helps align content properly. Extra styling like shadows, padding, and hover effects enhance the visual design.

Responsive Design

To make the website responsive, I used:
• Meta viewport tag
• Media queries
• Flexible units (%, rem, vh)
• Responsive images

These ensure proper display on laptops, phones, and tablets.

Challenges

Major challenges included:
• Responsive alignment issues
• Browser compatibility
• Spacing & layout management
• Implementing the hamburger menu

The hamburger menu still needs improvement, which I plan to fix later.

Conclusion

This project strengthened my understanding of HTML, CSS, Git, and GitHub. It helped me improve my design approach, debugging skills, and overall development workflow. This portfolio serves as a strong foundation for my future web development journey.
