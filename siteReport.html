Site Report

Overview

This website provides information on the entire web development learning process. During my web development adventure, I learned a variety of coding methodologies, design and debugging processes, and more.

Learning HTML and CSS

HTML stands for Hypertext Markup Language, which is used to create the basic structure of a website. CSS stands for Cascading Style Sheets, used to style the HTML structure. Throughout my learning experience, I learned about HTML, CSS, tags, elements, and CSS properties. In my portfolio, I used HTML5 and CSS3.

Learning Git and GitHub

Git is a distributed version control system used to track code changes, whereas GitHub is a cloud platform for hosting Git repositories. I learned commands like git clone, git push, git pull, git commit and concepts like repositories and branches.

Design Layout

I wanted my portfolio to look professional, so I chose minimal and clean colors. I used transitions and hover effects for a modern look. The website includes a header and multiple sections, with anchor tags connecting the pages.

Navigation Bar

The navbar is placed at the top of the website. It includes my logo and links to other pages. I used the flexbox display property to align items horizontally. Hover effects were added to enhance user experience.

Home Page

The homepage includes an introduction, an image, and a link to my projects. I used flexbox to arrange the layout, along with padding, margins, and box shadows. I also added hover opacity effect to the image.

Validation for index.html

The homepage HTML was validated to ensure clean structure and correctness.

My Projects

As a beginner, I do not have any completed projects yet. Therefore, I created placeholders using CSS grid layout. I used box shadows, border-radius, and subtle colors.

Contact Page

The contact page allows users to reach me via email. It includes my personal details, such as email, phone number, and address. Social media links are also included.

Responsive Design

Responsive design helps websites adjust to various devices and screen sizes. I used the viewport meta tag and media queries to ensure my website is responsive.

Challenges

I encountered several issues such as browser compatibility, responsive layout problems, and content organization. A major issue I couldn't solve was adding a hamburger menu for mobile view. It caused alignment problems, but I plan to fix it in the future.

Conclusion

Through this web development journey, I learned HTML, CSS, Git, and GitHub. This portfolio website reflects my learning and growth in web development.
